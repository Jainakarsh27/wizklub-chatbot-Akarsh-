# 🤖 WizKlub Smart STEM Assistant

A fully functional chatbot prototype designed to improve visitor engagement, qualify leads, and drive demo bookings for WizKlub.

---

## 📌 Project Overview

This chatbot is built to serve two primary user segments:

- 👨‍👩‍👧 Parents exploring STEM programs
- 🏫 Schools interested in partnership opportunities

The chatbot identifies user type, collects relevant details, qualifies leads, and encourages demo bookings through structured conversational flows.

---

## 🎯 Objectives

- Increase website engagement
- Reduce visitor drop-off rate
- Capture qualified leads
- Drive demo bookings
- Segment users (Parent / School)

---

## 🔄 User Flow Structure

### 👨‍👩‍👧 Parent Flow
1. Identify child’s grade
2. Capture interest area (Coding / Robotics / AI / Math)
3. Understand parent goal
4. Collect Name, Phone, Email
5. Provide Demo Booking CTA

### 🏫 School Flow
1. Capture designation (Principal / Teacher / Coordinator)
2. Ask student strength
3. Identify partnership interest
4. Collect contact details
5. Provide partnership scheduling CTA

## 📊 Lead Qualification Strategy

The chatbot assigns intent levels based on:
- Grade selection
- Interest in advanced STEM areas
- School size (for partnerships)

This helps prioritize high-intent leads.

- Lead Capture Rate
- Demo Booking Conversion Rate
- Drop-off Points
- High Intent Lead Percentage
